Датчик температуры и влажности

ДИ-ТВ-К65

Датчик имеет канальное исполнение для установки на воздуховод системы вентиляции. Передача показаний осуществляется по интерфейсу RS-485, протокол Modbus RTU. На корпусе имеются два гермоввода для удобного крепления кабелей. Зонд выполнен из нержавеющей стали. На плате расположен переключатель адреса. Имеется светодиодная индикация питания и передачи данных.
