Контроллеры Segnetics

Название Цена
Pixel 20 000 руб
Matrix 30 000 руб

tr - "table row". Используется для создания строки в таблице.

td - "table data". Используется для создания ячейки таблицы, которая содержит данные.

th - это сокращение от "table header". Используется для создания заголовочной ячейки в таблице.
